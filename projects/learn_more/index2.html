Modelling Molecular Transport Across Biological Barriers

This project focused on understanding how molecular properties influence transport across biological barriers, using the example of transplacental drug passage. The goal was to build a structured framework to analyse how molecules behave when crossing complex interfaces, combining data-driven approaches with physical intuition.

Problem

Predicting whether a molecule can cross a biological barrier is a complex problem. It depends on multiple interacting factors such as molecular size, polarity, and interaction with the surrounding environment. Existing approaches are often fragmented, with limited structured datasets available for analysis.

Approach

I built a structured dataset from scientific literature, combining heterogeneous data sources into a consistent format. This required data cleaning, feature engineering, and careful validation to ensure reliability.

On top of this dataset, I developed predictive models to analyse how molecular descriptors relate to transport behaviour. The work involved statistical modelling and machine learning techniques, but the focus remained on understanding the physical meaning behind the predictions.

Key Work

• Data collection and structuring from multiple scientific sources
• Feature engineering based on molecular properties
• Development and evaluation of predictive models
• Analysis of relationships between structure and transport behaviour

Outcome

The project resulted in a consistent dataset and a set of models capable of identifying patterns in molecular transport behaviour. More importantly, it provided insight into how molecular structure influences barrier permeability, linking data-driven results with physical interpretation.

Relevance

This type of work is directly related to broader problems in materials and biological systems, where transport, diffusion, and interaction mechanisms determine system behaviour. It reflects my general approach: combining data, modelling, and physical understanding to analyse complex systems.
