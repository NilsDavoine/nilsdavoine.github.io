Molecular Transport and Barrier Properties

Data modelling, molecular descriptors, and transport behaviour.

Modelling Molecular Transport Across Biological Barriers

This project focused on understanding how molecular properties influence transport across biological barriers, using the example of transplacental drug passage. The goal was to build a structured framework to analyse how molecules behave when crossing complex interfaces, combining data-driven methods with physical interpretation.

Problem

Predicting whether a molecule can cross a biological barrier is difficult. It depends on several factors, including molecular size, polarity, lipophilicity, and interaction with the surrounding environment. Existing data is often scattered across different scientific sources, making it hard to use directly for modelling.

My Work

I built a structured dataset from scientific literature and converted heterogeneous information into a consistent format. This involved data collection, cleaning, validation, feature engineering, and preparation of analysis-ready datasets.

I then developed predictive models to study how molecular descriptors relate to transport behaviour. The work used statistical modelling and machine learning, but the main focus was not only prediction. The goal was also to understand which molecular properties were linked to barrier crossing.

Technical Focus

• Data collection from scientific literature
• Cleaning and structuring of heterogeneous molecular data
• Feature engineering from molecular descriptors
• Predictive modelling using Python, R, SQL, and Scikit-learn
• Interpretation of structure-transport relationships

Outcome

The project produced a consistent dataset and a modelling workflow to analyse molecular transport behaviour. It helped identify patterns between molecular structure and passage across a biological barrier.

Relevance

Although this project was developed in a biomedical context, the reasoning is close to many problems in materials and membrane science. Transport, diffusion, and interactions are often controlled by molecular structure. This project trained me to connect data, molecular properties, and physical behaviour in a practical way.
