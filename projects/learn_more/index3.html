High-Performance Simulation Tools for Molecular Systems

This project comes from my PhD work in theoretical chemical physics, where I developed and used simulation tools to study large molecular systems. The goal was to make complex calculations more reliable, easier to reproduce, and better integrated into daily research workflows.

Context

Molecular simulations can become very expensive when systems grow in size or when long-range interactions need to be treated carefully. In my work, this was especially important for polymers, carbon-based structures, and large systems where van der Waals interactions play a major role.

My Work

I worked with simulation tools such as LAMMPS, GROMACS, and custom code. I also developed Fortran-based tools for analysis and simulation workflows. The focus was on performance, numerical reliability, and making the code usable in a research group setting.

Technical Focus

• Simulation and analysis of polymers, carbon chains, and molecular systems
• Development of custom scientific code in Fortran and Python
• Workflow design for reproducible calculations
• Analysis of interaction energies, forces, and structural behaviour
• Integration of tools into existing research workflows

Why It Matters

The aim was not only to produce results, but to build tools that could be trusted and reused. In scientific work, a method is only useful if it is clear, stable, and understandable by others. This experience shaped the way I now approach technical problems: first understand the physics, then build the tool around it.

Relevance

This work is directly connected to materials modelling and product-oriented R&D. It trained me to think about molecular interactions, structure-property relationships, and the practical limits of simulation when working with large systems.
